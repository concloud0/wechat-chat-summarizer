# 隐私与风险说明

## 数据处理

- `local` 模式只在本机解析和生成规则摘要，不上传聊天内容。
- `deepseek` 模式会将筛选后的聊天内容发送到所配置的 DeepSeek API。
- 使用 DeepSeek 前，请确认聊天参与者、所在组织及适用法规允许上传相关内容。
- 建议先使用日期和成员筛选，并移除身份证号、账号、地址、商业秘密等敏感信息。

## API Key

- API Key 保存在 `%LOCALAPPDATA%\WeChatChatSummarizer\settings.json`。
- 密钥使用 Windows DPAPI 当前用户作用域加密，不以明文写入配置文件。
- 密文通常只能由同一台电脑上的同一 Windows 用户解密。
- 删除 `%LOCALAPPDATA%\WeChatChatSummarizer` 可清除 API Key 和全部应用设置。

## 微信会话直读

- 微信直读依赖外部工具 `wechat-cli`，本发布包不包含该工具。
- 初始化和读取过程会访问本机微信数据。请自行确认该工具的来源、许可和适用性。
- 文本文件摘要功能不依赖 `wechat-cli`。

## 试用版说明

- 本软件为个人工具试用版，按现状提供，不保证适用于所有聊天导出格式或业务场景。
- AI 摘要可能遗漏、误解或错误归纳信息，重要结论必须回看原始聊天记录。
- 本版本未进行 Authenticode 代码签名，Windows 可能显示“未知发布者”。
